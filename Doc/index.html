Introduction

This Web application allows to observe the evolution of the digital cats as well as the owners.You can visualize the most important owners,newest "cats" and their family tree.

General design

  • single page application ( WIP )
  • provides various options structured as follows:
    • A user profile page where is displayed all the information about that user /current user
    • A cat profile page where is displayed all the information about that cat
    • A page with newest cats
    • A page with top owners depending on the number of the cats the own

User interaction

The user will interecat with the page through any modern browser with a valid source of internet connection

Web development tools and technologies

  • The web app is coded using Visual Studio Code
  • The code is then run and tested using Google Chrome
  • Then we upload the code modifications to GitHub
  • Communications and task management is done through Trello
  • Technologies used: HTML5, CSS, JavaScript
